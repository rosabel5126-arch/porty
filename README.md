# Rosabel's Portfolio

A responsive portfolio website showcasing work and experience.

## Features

✅ **Mobile-Responsive Design** - Optimized for all device sizes
✅ **Embedded Canva Portfolio** - Beautiful design preview
✅ **Direct PDF Downloads** - Portfolio and Resume available for download
✅ **Modern UI** - Clean and professional interface

## Project Structure

```
porty/
├── index.html          # Main portfolio page
├── assets/
│   ├── portfolio.pdf   # Portfolio document (placeholder)
│   └── resume.pdf      # Resume document (placeholder)
└── README.md          # This file
```

## Setup Instructions

1. **Clone the Repository**
   ```bash
   git clone https://github.com/rosabel5126-arch/porty.git
   cd porty
   ```

2. **Add Your PDF Files**
   - Replace `assets/portfolio.pdf` with your actual portfolio
   - Replace `assets/resume.pdf` with your actual resume

3. **Deploy with GitHub Pages**
   - Go to repository Settings → Pages
   - Select "Deploy from a branch"
   - Choose `main` branch and `/root` folder
   - Your portfolio will be live at: `https://rosabel5126-arch.github.io/porty/`

## Customization

### Update Portfolio Link
Edit the Canva embed link in `index.html`:
```html
<iframe src="https://www.canva.com/design/YOUR_DESIGN_ID/YOUR_CODE/view?embed"></iframe>
```

### Modify Styling
All CSS is in the `<style>` section of `index.html`. You can customize:
- Colors
- Fonts
- Spacing
- Responsive breakpoints
