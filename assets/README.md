# Portfolio Assets

This folder contains downloadable PDF files for the portfolio.

## Files

- **portfolio.pdf** - Main portfolio document (placeholder)
- **resume.pdf** - Resume/CV document (placeholder)

## How to Add Your PDFs

1. Replace `portfolio.pdf` with your actual portfolio PDF
2. Replace `resume.pdf` with your actual resume PDF
3. Both files will be directly downloadable from the portfolio website

## File Size Recommendations

- Portfolio PDF: Less than 5MB for optimal download speed
- Resume PDF: Less than 2MB recommended
